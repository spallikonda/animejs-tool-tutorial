Learn & Build with Anime.js

A tool and tutorial site for mastering Anime.js animations. Compose effects, see live previews, and export code.

Start Animating

Basics

Learn to animate CSS properties like translate, scale, opacity.

anime({ targets: '.demo', translateX: 250 });

Timelines

Sequence animations with timelines.

const tl = anime.timeline(); tl.add({ targets: '.demo', scale: 2 });

Staggering

Delay animations across elements.

anime({ targets: '.demo', delay: anime.stagger(100) });

Easings

Use built-in easings like easeInOutQuad.

anime({ targets: '.demo', rotate: 360, easing: 'easeInOutSine' });

SVG Paths

Animate SVG paths with stroke-dashoffset.

anime({ targets: 'path', strokeDashoffset: [anime.setDashoffset, 0] });

Playback Controls

Pause, play, reverse, and seek animations.

anime({ targets: '.demo', autoplay: false }).play();
